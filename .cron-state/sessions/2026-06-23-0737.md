## Five features shipped

1. **`regimen-snapshot-archive-history-rollup-csv-export-merge-anonymise-key-rotate-bulk-csv-export-per-class`** (`2e13caf`)
   - Group bulk-rotation chains by drug class then CSV export so a
     cardiology clinic can filter the chain to just the meds in its
     remit (statin.csv + beta-blocker.csv + ace-inhibitor.csv as
     three separate sheets instead of the master 47-column sheet).
   - patientClasses: `ReadonlyMap<patientId, ReadonlySet<DrugClassCode | string>>`
     (tolerates custom non-DrugClassCode strings).
   - classesToEmit restricts the subset; includeUnclassified (ON by
     default) captures patients with no matching class.
   - basenameTemplate / unclassifiedBasename for file basenames.
   - Manifest CSV columns: classCode, basename, patientCount, transitionCount.
   - listAnonymiseKeyRotateBulkCsvExportPerClassFiles flattens
     chains + transitions + manifest into a file-entry array.
   - Transition patientCount equals FILTERED patient count (audit
     numbers are honest per class).
   - 38 tests.

2. **`dose-export-csv-import-roundtrip-validator-summary-text-slack-thread-batcher-quiet-hours-calendar-html-per-cell-edit`** (`8437a62`)
   - Wraps each calendar cell in `<a href>` so a channel admin can
     click "Wed" and jump to the override editor pre-scoped to
     Wednesday.
   - editUrlTemplate placeholders: {day}, {dayLabel}, {rule}
     (URI-encoded before substitution).
   - isCellEditable predicate selectively suppresses the anchor
     on a cell.
   - buildAriaLabel override; default
     "Edit quiet hours for {dayLabel} (currently {ruleLabel})".
   - openInNewTab for target="_blank" rel="noopener".
   - editLinks parallel array exposes resolved hrefs for hosts
     that only need URL shapes.
   - 35 tests.

3. **`followup-digest-text-html-bundle-i18n-multi-locale-cron-batcher-html-mailer-bcc-tier-policy-coverage-report-html`** (`a938449`)
   - Dashboard-ready HTML render of the JSON-friendly coverage
     report.
   - Section structure: headline card (envelope + BCC + dominant
     tier), tier-distribution bars (routine grey, actionable
     amber, critical red, width = ratio), top fan-out table (sorted
     count DESC, monospace addresses), warnings panel (red border,
     light-red background), escalation-only addresses list.
   - topFanoutRowLimit defaults 10 (0 suppresses).
   - suppressWarnings for admin overlays.
   - tierLabels override for localised labels.
   - wrapHtmlDocument for browser print.
   - HTML-escapes every address + warning + label against XSS.
   - 36 tests.

4. **`refusal-reason-suggest-i18n-rollup-html-print-cover-sheet-binder-spine-batch-csv-manifest-pivot`** (`be242e0`)
   - Pivot the spine-row manifest into a sheet-row pivot for the
     printer-cassette-loader workflow.
   - One row per sheet, columns expand to one per position
     (pos_1, pos_2, ..., pos_N where N = capacity).
   - Position cells in row-major order; empty positions are bare
     empty cells by default (configurable via emptyPositionPlaceholder).
   - positionColumnTemplate (default 'pos_{n}') overrides header pattern.
   - includeDateLabelInPosition formats cells as
     'patientName (dateLabel)'.
   - detectPartialSpineSheets returns sheets with spineCount < capacity.
   - source manifest exposed for callers wanting both views.
   - 37 tests.

5. **`prescriber-contact-card-emergency-card-pdf-two-up-watermark-roster-toc-html-anchored-back-to-top`** (`87dbec7`)
   - Adds a return-path to the anchored TOC: injects
     `<a id="${tocTopAnchorId}" tabindex="-1">` at the top of the
     .toc-wrapper section, and exposes a backLinkByCardIndex Map
     of pre-rendered "Back to TOC" link fragments.
   - tocTopAnchorId defaults to `${tocPrefix}-top` (per-prefix
     disambiguation for multi-TOC host pages).
   - backLinkLabel + backLinkClassName overrides for i18n + styling.
   - buildBackLinkAriaLabel default
     "Return to table of contents from {displayName} card".
   - tabindex=-1 on top anchor: programmatic JUMP target without
     natural tab-sequence focus.
   - HTML-escapes labels, aria-labels, prescriber names against XSS.
   - 32 tests.
