## Shipped (5)

1. **today-section-progress-bar** (4ad7a6e) — Each Morning/Afternoon/Evening/
   Night section header on /today already showed a "M of N taken" count chip;
   it now also carries a thin progress bar so a glance reads progress without
   parsing the number: a sage taken segment plus a muted amber skipped sliver in
   the width left after taken, with a proper aria progressbar role + label. New
   `lib/section-progress.ts` (sectionProgress + sectionProgressLabel) composes
   lib/part-of-day's PartOfDayCounts; fractions clamp 0..1 and taken+skipped can
   never overflow the track. 13 tests.

2. **medications-sort-persist** (c907846) — The /medications Name / Lowest
   supply / Soonest refill sort reset to Name on every reload; it now persists
   to localStorage and restores on mount, matching the density toggle, run-out
   grouping flag, and the refills sort already on the page. New
   `lib/med-sort-pref.ts` (storage key + default + normalize/parse/serialize
   guards); unknown stored key falls back to Name. 12 tests.

3. **caregivers-expiring-headline** (c9b991a) — The caregivers list already
   pills individual soon-to-lapse rows; this adds a header-level amber tally
   chip under the title ("2 expiring soon" / "1 expired" / "2 expiring soon,
   1 expired"), null when nothing's soon/expired. Counted across ALL shares (not
   the filtered view) so the header stays honest while searching. Added
   `summarizeExpiry` + `expiringHeadline` to `lib/caregiver-expiry`. 8 new tests
   (24 in suite).

4. **dashboard-strip-tooltip-dates** (91e8d87) — The dashboard two-week strip
   cells only said "Prior / Current window avg" on hover; each now leads its
   title with the real calendar date (today minus its index): "Today, Jun 26 —
   current-window average 87%", "Yesterday, Jun 25 — ...", "Sat, Jun 13 —
   prior-window average 80%". Stays honest: the date is exact but the percentage
   is still explicitly a WINDOW AVERAGE (continues the tick-37 trend-series
   cleanup), never implying we know that single day's real adherence. New
   `lib/strip-dates.ts`; locale-stable date tables, injectable now, clamps so it
   can never name a future day. 14 tests.

5. **dashboard-streak-ring-accent** (7c8c5cc) — The dashboard streak tile +
   Two-week-pulse capsule were always sage regardless of streak length; they now
   read at a glance: 0 days neutral, 1-6 days amber (building), 7+ days sage
   (going strong). The stat tile also gains a milestone nudge ("2 days to a
   week" -> "going strong"). New `lib/streak-tone.ts` (streakTone / streakAccent
   / streakToneVar / daysToStrong) — a SHARED classifier the history streak
   callout can adopt too (Tier 2B #356 queued for that). 14 tests.
