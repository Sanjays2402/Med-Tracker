## Highlights

- **Bulk key-rotation** is the first multi-epoch HMAC rotation chain in the package. Walks N-1 transitions in parallel via Promise.all over the single-step primitive, stitches them into per-patient pseudonym chains across every epoch. Three lookup helpers (first-to-last, arbitrary-epoch-to-epoch, without-original-ids) cover typical audit-trail questions. Use case: clinic auditing a decade of annual HMAC secret rotations needs the full chain in one call instead of N hand-stitched transitions.
- **Calendar quiet-hours** is the first per-day-of-week quiet-hours module. Three override kinds (window / quiet-all-day / no-quiet-hours), Intl.DateTimeFormat for day resolution in the channel's timezone (Friday 23:30 UTC in PT is correctly Friday PT). Convenience builder for the canonical "weekends quiet, weekdays 22:00-07:00 PT" config.
- **BCC tier policy** is the first tier-aware BCC overlay. Three canonical tiers (routine/actionable/critical), default heuristic classifier on text + subject phrases, per-tier + per-caregiver scope correctly composed. unusedDestinations flags declared-but-never-matched addresses (common misconfiguration signal).
- **Spine batch** is the first multi-spine sticker-sheet layout. CSS grid sized to sheet + spine + gap; us-letter / a4 / custom presets; forceColumns / forceRows for fixed-stock sticker paper; multi-sheet pagination with page-break-before on each after the first. Use case: clinic printing 47 binder spines after a Q3 review uses 2 sticker sheets instead of 47 print jobs.
- **TOC HTML** is the first HTML/CSS companion to the PDF TOC. Four @page presets + custom inch dimensions; renamed htmlPageSize to dodge the underlying PDF's pageSize ('letter'|'a4'). wrapHtmlDocument toggles full-document vs fragment mode. Watermark as fixed-position overlay matching the PDF's text + visual weight.
