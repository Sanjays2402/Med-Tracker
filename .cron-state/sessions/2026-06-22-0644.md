   underlying digest math runs ONCE per patient. Unknown-locale policy
   supports fallback-en / skip / error. Coverage telemetry includes
   silentCaregiverIds, skippedCaregiverIds, localeUsage. 17 tests.

   variant of i18n-rollup-html. No interactive controls (paper signoff
   bubble), paginated with page-break-after:always + page-break-inside:
   avoid, repeating header on every page, print-friendly serif font,
   coverage strip on page 1 only. Sets the pattern for the upcoming
   validator-html-print (#142). 23 tests.
5. **prescriber-contact-card-emergency-card-pdf-two-up-watermark** —
   diagonal overlay watermark spanning BOTH slots of the landscape
   two-up sheet. Presets: draft / verified (with date) / icu-copy /
   do-not-fax / controlled / custom. Page-centred, -30° rotation,
   96pt bold gray-400 at 0.18 opacity by default; every dimension
   overridable. Multi-page builder LOCKS watermarkVerifiedAt across
   the run so a midnight rollover can't produce a mixed-date stack.
   25 tests.
