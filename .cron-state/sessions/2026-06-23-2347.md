1. `dc76a45` — **command-palette-cmd-k** (apps/web/components/CommandPalette.tsx)
   Linear/Raycast-style ⌘K palette. Fuzzy search across nav, theme
   actions, and the user's medication list. Subsequence scoring
   (prefix > substring > consecutive chars), title weighted 1.5x over
   subtitle. ⌘K / Ctrl+K / `/` opens, Esc closes, ↑↓ navigates, ↵ runs,
   Home/End jump. Lazy-loads meds on first open. Mouse hover updates
   active index. Auto-scroll keeps active row visible. Discoverable
   topbar "Search or jump…" hint badge with mac/non-mac shortcut glyph
   that dispatches a synthesized Cmd+K to open the same code path.

2. `8905b13` — **toast-notifications** (apps/web/components/Toast.tsx +
   today/page.tsx + globals.css)
   <ToastProvider> mounted in (app)/layout. useToast hook from anywhere.
   Four kinds (success / error / warning / info), each with semantic
   icon + tone from existing design tokens. Auto-dismiss with hover-pause
   (timer freezes on mouseenter, resumes from remaining time on
   mouseleave). Inline action button (e.g. Undo) that runs callback +
   dismisses. Dedupe by externalId. Slide-in/slide-out keyframes;
   prefers-reduced-motion respected. Wired into today's dose-take +
   dose-skip flow with an Undo action that calls undoDose.

3. `822d699` — **adherence-ring-widget** (apps/web/components/AdherenceRing.tsx
   + dashboard/page.tsx)
   Pure SVG animated donut. Stroke length tweens to target percent via
   requestAnimationFrame easeOutCubic. Starts from 0 so the ring sweeps
   in on first paint. prefers-reduced-motion jumps to target but keeps
   theme-flip stroke-colour transitions. 0/25/50/75 milestone ticks.
   Tone auto-derives from percent. Center slot is a render prop. Replaces
   the dashboard's static sine-wave 14-day grid with a coherent ring +
   real-percent-derived 14-day grid; today's cell carries a sage outline.

4. `462c2f1` — **keyboard-shortcuts-overlay** (apps/web/components/KeyboardHelp.tsx
   + layout.tsx)
   Linear-style ? cheat sheet plus global shortcut router. Leader
   sequences: G then D/T/M/S/R/H route to dashboard / today / medications
   / schedule / refills / history (1.4-second window). N opens new med.
   T toggles theme (clicks the existing button — useTheme stays single-
   source-of-truth). Esc closes overlays. All shortcuts pause when
   focus is in a text field. Mac/non-mac modifier glyph swap (⌘ vs Ctrl)
   on every kbd block. Topbar gets a "?" chip that dispatches a
   synthesized "?" KeyboardEvent.

5. `a28108a` — **history-page-heatmap** (apps/web/app/(app)/history/page.tsx)
   Full page redesign. 26-week (~6 months) GitHub-contributions-style
   heatmap. Each cell 14x14px. Sage progression for healthy days
   (95+ / 85+ / 70+ stops), warm amber for shaky (50+), coral for rough
   (<50). Today's cell outlined in sage. Weekday labels on left rail
   (every other row). Month labels above each month's first column.
   Hover/focus scales the cell up 1.25x with 1.5x outline and surfaces a
   detail row underneath (full date, % on schedule, dose count,
   qualitative chip). Three stat tiles (6-month avg, perfect days, rough
   days). "Less … more" legend mirroring the cell ramp. "Recent days"
   list of last 7 entries below with date-pill glyph tinted to tone +
   inline % + qualitative chip + ArrowRight to per-day page. Falls back
   to deterministic per-iso-date percentages until the API returns real
   history.
