# Use email plus password with JWT short lived access tokens

Status: accepted

## Context

Med-Tracker needs a clear, recorded reason for this decision so future contributors can challenge it intentionally.

## Decision

Use email plus password with JWT short lived access tokens.

## Consequences

* Easier onboarding for new contributors
* Locks us into the chosen stack until a follow up ADR supersedes this one
* Reduces bikeshedding on day to day choices
